# 彩票查询

微信原生小程序 + 云开发版本，支持超级大乐透、双色球开奖结果查询。

## 功能

- 大乐透、双色球开奖结果查询（支持切换）
- 首次访问时自动补拉最近一期，避免空库首屏无数据
- 历史开奖按期号、年份、日期查询
- 上月同日、上六个月同日、上一年同日查询
- 目标日期无开奖时返回相邻前后期开奖
- 开奖日定时同步最新结果
- 数据说明和审核说明页面

## 彩种信息

| 彩种 | 号码规则 | 开奖日 | 数据起始 |
|------|---------|--------|----------|
| 大乐透 | 5前区(1-35) + 2后区(1-12) | 周一、三、六 | 2015-01-03 |
| 双色球 | 6红球(1-33) + 1蓝球(1-16) | 周二、四、日 | 2003-02-23 |

## 技术方案

- 小程序：微信原生小程序
- 后端：微信云开发云函数
- 数据库：微信云数据库
- 数据源：Huiniao 免费开奖接口
- 官方说明：中国体育彩票开奖结果为最终依据

## 目录

```text
miniprogram/              小程序前端
cloudfunctions/lottery/   统一云函数
```

## 需要你填写

请在以下位置填写你的配置：

1. `project.config.json` → 填写你的 AppID
2. `miniprogram/app.js` → 填写你的云开发环境 ID

## 云数据库集合

在云开发环境中创建：

```text
dlt_results     大乐透开奖结果
ssq_results     双色球开奖结果
sync_logs       同步日志
```

## 云函数接口

统一函数名：`lottery`

参数 `lotteryType`：`dlt`（大乐透）或 `ssq`（双色球）

支持 action：

```text
getLatest      获取最近一期
queryHistory   历史查询
getDetail      查询详情
getRelative    查询上月/上六个月/上一年同日
syncLatest     同步最新一期
syncHistory    分页初始化历史数据
```

## 同步策略

云函数触发器配置在：

```text
cloudfunctions/lottery/config.json
```

当前策略：

```text
大乐透：每周一、三、六 21:25-21:50 每分钟执行一次
双色球：每周二、四、日 21:25-21:50 每分钟执行一次
```

函数内部还会再次判断是否处于开奖窗口，避免误触发。

## 本地打开

1. 打开微信开发者工具
2. 导入本目录
3. 填写你的 AppID
4. 开通云开发并填写环境 ID
5. 上传并部署 `cloudfunctions/lottery`
6. 调用 `syncHistory` 初始化历史数据（需指定 lotteryType）

详见 [DEPLOYMENT.md](./DEPLOYMENT.md)。
